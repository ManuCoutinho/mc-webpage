## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/ManuCoutinho/oishi-meals

# Access
$ cd oishi-meals

# Install dependencies
$ npm install

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:5173>
```
